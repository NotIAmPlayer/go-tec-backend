# Guides
## How to run backend locally
- Clone this repository.
- Import dependencies for this repository. (use `go mod download`, `go mod tidy`, you can look it up)
- Run backend with `go run .` in the terminal.
- Modify `.env` file if needed.

## How to build backend
- Use `go build main.go` in the terminal. (You can name the build file with `-o [name]` placed after the command, like `go build main.go -o [name]`) - the usual filename is `go-tec-backend`
- Run the backend application in the server (you can use cron, nohup, etc.)
